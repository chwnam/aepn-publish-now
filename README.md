# AEPN Publish Now

AEPN (attorneyengineer-publish-now) Publish Now 플러그인은
워드프레스 블록 에디터 내에서 `Ctrl+Alt+S`를 누르면 바로 발행 상태로 넘어가게 해 주는 플러그인입니다.
